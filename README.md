# Ping-Command-Connectivity-Testing
A project explaining the working and use of the PING command for network connectivity testing.

## Project Overview
This project explains the working of the PING command, a basic network diagnostic tool used to test connectivity between devices in an IP network.

## Objectives
- Understand how the PING command works
- Test network availability
- Measure latency and packet loss
- Diagnose basic network issues

## Tools Used
- ICMP Protocol
- Windows Command Prompt / Linux Terminal

## Project Contents
- PPT explaining the PING command
- Command examples
- Output interpretation
